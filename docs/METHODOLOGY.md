# Methodology summary

The review chain tests whether C2025AUG / 2025-08-04 remains visible as a multi-device, multi-layer platform-state cluster after progressively stricter checks.

Model axes:
1. cross-strict raw/result agreement
2. raw-only external critical survival
3. Trial / A-B / cohort overlap
4. direct triald source-time overlap
5. trust graph lineage shadow
6. backup / Manifest inheritance shadow
7. proximity vs cloud separation
8. evidence preservation / suppression pressure
9. normal/control stress reduction

Interpretation:
- `supported` means local artifact consistency, not cause.
- `shadow` means local artifact trace/edge, not server-side proof.
- `review` means retained as an expert-review target, not confirmed malicious activity.
